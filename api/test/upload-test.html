Guia de Solução de Problemas de CORS

  • API NestJS bloqueando CORS: O testador precisa que a API aceite requisições de outras origens. Certifique-se de que app.enableCors() foi adicionado no main.ts da sua API.
  • R2 / S3 Bucket bloqueando CORS no PUT: Por padrão, buckets da AWS S3 e Cloudflare R2 rejeitam uploads diretos do navegador devido a políticas de segurança.
  • Configuração de CORS no Cloudflare R2: Adicione esta regra nas configurações de CORS do seu bucket: 
    [
  {
    "AllowedOrigins": ["*"],
    "AllowedMethods": ["PUT", "GET"],
    "AllowedHeaders": ["content-type"],
    "ExposeHeaders": []
  }
]
