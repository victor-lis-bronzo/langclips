# Glossário

- **Clip:** Trecho de vídeo/áudio selecionado pelo usuário para ser usado no exercício.
- **Sessão Volátil:** Sessão de exercícios que não é salva no banco de dados, apenas armazenamento local.
- **Deck:** Coleção de vídeos selecionados pelo usuário para serem usados no exercício.
- **Ditado:** Exercício de digitação onde o usuário deve digitar o que ouve.
- **Lacuna:** Palavra da frase que o usuário deve preencher.
- **Múltipla Escolha:** Exercício de múltipla escolha onde o usuário deve escolher a palavra correta para preencher a lacuna.
